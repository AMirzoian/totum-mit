# Расширяемый open source конструктор, для быстрой разработки внутренних продуктов

> — Open source конструктор внутренних ИТ-продуктов для компаний 20-300 человек.

> — Упрощает реализацию скучных для разработчиков, но необходимых бизнесу задач по структуризации данных и взаимодействию сотрудников.

> — Реализован на PHP и PostgreSQl, но может расширяться на любом серверном языке.

> — Базовая версия распространяется по [лицензии MIT.](https://totum.online/licenzirovanie)

**База данных с готовым типовым интерфейсом, структурой,  логикой транзакций, внутренними скриптами (PostgreSQL).**

Типовые модули (текст, числа, файлы, ссылки, списки, даты, данные, кнопки, комментарии итд).

Распределение прав (роли, пользователи, иерархия доступа, доступ по условиям).

Инструменты отладки (расчет значений, выполнение действий, условия форматирования, нагрузка, время выполнения).

Задачи по расписанию (подключение cron на выполнение внутренних задач).

Свободная доработка (разрабатывайте модули на любом языке, подключайте в единый интерфейс, добавляйте функции).

## Распространение коммерческих продуктов

Решения созданные на Тотум могут переносится из одной схемы в другую со следующими лицензиями:
	
`Free/MIT` — копирование без ограничений (для копирования собственных проектов или их частей и их свободного распространения). **Доступно на текущий момент!**

`Totum` — лицензирование через сервер лицензий Тотум.

	В разработке:
	    
		— Разработчику требуется зарегистрироваться на сервере лицензий Тотум.
		— Выпуск лицензии платный.
		— Выдача лицензии оплачивается разработчиком по тарифу по окончании отчетного периода.
		— Возможны срочные и бессрочные лицензии.

`Custom` — лицензия подтверждается на сервере лицензий разработчика.
      
	В разработке:
	  
		— Требуется купить лицензию на TOTUM custom-server (лицензия Totum).

`Local` — автономная сборка с собственным сервером лицензий, зашифрованная отличным от общественной версии ключом.
